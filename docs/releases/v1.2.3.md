# Cortex v1.2.3 Release Notes

Date: 2026-03-05
Tag: `v1.2.3`

Patch release focused on reason-output quality and digest reliability.

## What's New

### Reason Output Passthrough
`enforceResponseQualityContract()` no longer injects hardcoded boilerplate content.

This prevents generic normalized text from masking actual model output quality in downstream digest/reporting pipelines.

### Cleaner Observability
By preserving model output, operators can now see true model behavior and let external quality gates decide:
- pass (use LLM output)
- retry/fallback (deterministic summary)

This makes failures diagnosable instead of silently “normalized.”

### Tests Updated
Reason quality-contract tests now validate passthrough behavior for this path.
