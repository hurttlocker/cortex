## Upgrade Notes

```bash
# Homebrew
brew update && brew upgrade cortex-memory

# Manual (macOS Apple Silicon)
curl -sSL https://github.com/hurttlocker/cortex/releases/download/v0.6.0/cortex-darwin-arm64.tar.gz | tar xz
sudo mv cortex /usr/local/bin/

# Try the new import filters

# Reimport with auto-inference
cortex import ~/notes/ --recursive --extract  # edges created automatically
```
