# Cortex v0.7.0 Release Notes

Date: 2026-02-24
Tag: `v0.7.0`

## Summary

`v0.7.0` is the connector and graph-UX release. The connect pipeline is now fully test-covered end-to-end, three new connectors are available (Discord, Telegram, Notion), and the graph explorer now supports practical search/filter/inspection workflows for daily use.

- **Telegram connector** (`internal/connect/telegram.go`)
  - Bot API `getUpdates` polling with chat filtering and incremental time-based sync
  - Message type support: text, photo captions, documents, forwarded/reply context
- **Notion connector** (`internal/connect/notion.go`)
  - Workspace search + page block extraction + optional database row import
  - Notion block-to-markdown conversion for headings/lists/code/callouts
